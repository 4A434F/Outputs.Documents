# BGOW0044 Document 1044 Mapping Report

Document `1044` is the CTAD client pre-contentious cancellation letter with pending receipts/value.

This report has two sections:

1. the fast mapping;
2. the validation tables that show where each decision can be checked in the legacy files and current .NET implementation.

## Source Files

| Source | Purpose |
|---|---|
| [`BGOW0044_PGOC114A`](../BGOW0044_PGOC114A) | BGOW0044 copybook structure |
| [`PGOC114A.txt`](../PGOC114A.txt) | COBOL program that evaluates the input and writes layouts |
| [`PFX014.txt`](../PFX014.txt) | Page/layout definition for generated fields |

## Fast Mapping

### Selection / Evaluation Fields

- `COD-DOCUMENTO = CTAD` and `VLR-DIVIDA != 0`.
- Legacy selection moves `1044` to `CODBAR-IDENTIF-DOCUM`.
- Legacy layout -> `X014CTADT2 / X014CTADTB`.

### Displayed Fields

#### Address Window

Mapping:

- `CL-NOME` -> `BGOW0044Contract.Client.Name`
- `CL-NOME` -> `BGOW0044Contract.Client.Address.Name`
- `CL-MORADA1` -> `BGOW0044Contract.Client.Address.Line1`
- `CL-MORADA2` -> `BGOW0044Contract.Client.Address.Line2`
- `CL-LOCALIDADE` -> `BGOW0044Contract.Client.Address.Locality`
- `CL-CPOSTAL` -> `BGOW0044Contract.Client.Address.PostalCode`
- `CL-CPOSTAL-DESC` -> `BGOW0044Contract.Client.Address.PostalCodeDescription`
- `CL-CPAIS` -> `BGOW0044Contract.Client.Address.CountryCode`
- `CL-CPAIS-DESC` -> `BGOW0044Contract.Client.Address.CountryDescription`
- `NR-CLIENTE` -> `BGOW0044Contract.Policy.ClientNumber`
- `CL-LOCATION-REF` -> not currently modeled directly

Rendered by:

- LetterLayout Address or future agent address component

#### Subject Block

Mapping:

- `CL-NOME` -> `BGOW0044Contract.Client.Name`
- `CL-NIF` -> `BGOW0044Contract.Client.TaxIdentificationNumber`
- `NR-APOLICE` -> `BGOW0044Contract.Policy.PolicyNumber`
- `DESCR-RAMO-PROD` -> `BGOW0044Contract.Product.BranchProductDescription`
- `risk text from UR-TIPO and related risk fields` -> `BGOW0044Contract.RiskUnit.Description`
- `NR-AGENTE` -> `BGOW0044Contract.Policy.AgentNumber`

Rendered by:

- LetterLayout Subject / SubjectDetails

#### Body Variables

Mapping:

- `DT-ANULACAO` -> `BGOW0044Contract.CancellationDate`
- `VLR-DIVIDA` -> `BGOW0044Contract.DebtAmount`
- `receipt rows` -> `BGOW0044Contract.ReceiptDetails`

Rendered by:

- document body/list paragraphs

#### Trace / Signature Context

Mapping:

- `NR-APOLICE` -> `BGOW0044Contract.Policy.PolicyNumber`
- `generated document id` -> `BGOW0044Contract.DocumentIdentification`
- `USERID` -> trace text source in legacy; current templates use static or layout-level trace data
- `COD-ORIGEM` -> trace text source in legacy; current templates use static or layout-level trace data
- `selected CTAD signer` -> `static SignatureComponent configuration`

Rendered by:

- LetterLayout TraceText / SignatureComponent

## Legacy Layout / PDF Evidence

Uses CTAD formatter `V300-FRMT-CTAD-CL` for client output and `V400-FRMT-CTAD-AG` for mediator output. Layout evidence: `X014CTADT2 / X014CTADTB`.
