## Fast Mapping

### Selection / Evaluation Fields

- `COD-DOCUMENTO = CDC1`.
- `DT-SINISTRO` is not empty.
- `COD-RAMO(1:4) = 0601` switches to the automotive layout variant only.
- Legacy layout -> `X014CDC1T1` or its automotive variant.

- Selection-only legacy fields should be modeled only if runtime selection needs to reproduce COBOL selection from raw input.
