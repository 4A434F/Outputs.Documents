# BGOW0044 Document 1001 Mapping Report

Document `1001` is a courtesy letter to the mediator generated as a side output from the CTAD flow when `NR-AGENTE` is present.

This report has two sections:

1. the fast mapping;
2. the validation tables that show where each decision can be checked in the legacy files and current .NET implementation.

## Fast Mapping

### Selection / Evaluation Fields

- `COD-DOCUMENTO = CTAD`.
- `NR-AGENTE` is not empty.
- Legacy CTAD flow temporarily moves `1045` for the mediator cancellation copy and also calls `T200-CARTA-CORTESIA`, which sets document id `1001` for the courtesy letter.

- `AG-CPOSTAL-DESC` -> not currently modeled as an agent address object
- `AG-CPAIS` -> not currently modeled as an agent address object
- `AG-CPAIS-DESC` -> not currently modeled as an agent address object
- `NR-AGENTE` -> `BGOW0044Contract.Policy.AgentNumber`

#### Courtesy Body

Mapping:

- `DT-EMISSAO` -> `BGOW0044Contract.IssueDate`
- `COD-ORIGEM`, generated document id, mediator number -> trace/id data

Rendered by:

- future `1001` courtesy template
